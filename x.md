### ЭТАП 2: Основной вход (Telegram Mini App)

Пользователь нажимает кнопку `[📱 Открыть Mini App]`. Приложение считывает `X-Telegram-Init-Data`, бэкенд на лету проверяет `telegram_id` по базе данных и сразу открывает личный кабинет без каких-либо текстовых инвайт-ключей.

```text
   ИНТЕРФЕЙС TELEGRAM MINI APP
  ──────────────────────────────────────────────────
   Maksim  •  Доступен: 30 дн. Макс 3 устройства
  
   Данные для входа через браузер (при ЧС):
  • Логин:  login
  • Пароль: xT9#mP2q сменить
  ──────────────────────────────────────────────────
  ➕ Новое подключение:
   [ Имя профиля: pc, ioc...    ]
   [ ⚡ Создать ]
  ──────────────────────────────────────────────────
  📱 Активные профили:
  • HomePC   [Ключ ] [ 💾 Скачать ] [ 🔍 QR-код ]
  ──────────────────────────────────────────────────

```

* При клике на `[⚡ Создать]` отправляется асинхронный POST-запрос на `/connect`. Так как пользователь валидирован мессенджером, бэкенд запрашивает у AmneziaWG новый `peer_id`, сохраняет его в таблицу `profiles` и выводит пользователю строку для импорта и QR-код.

## V. ЛОГИКА ОГРАНИЧЕНИЯ ДОСТУПА ПРИ ИСТЕЧЕНИИ СРОКА

Фоновый воркер (`workers.py`) запускается с заданным интервалом и выполняет мониторинг базы данных.

1. **Заморозка на сервере:** Если текущее время сервера превышает сохраненный `subscription_ends_at`, воркер меняет флаг на `is_premium = 0` и вызывает функцию `amnezia.revoke_user_access()` для всех `peer_id`, принадлежащих данному пользователю. Доступ к туннелированию физически блокируется.
2. **Блокировка интерфейсов:** При попытке открыть Mini App или обновить страницу веб-версии в браузере, бэкенд отдает единый блокирующий экран:

```text
🖥 / 🌐 ЭКРАН ПОЛНОЙ БЛОКИРОВКИ (ДОСТУП ИСТЕК)
──────────────────────────────────────────────────
⚠️ Доступ ограничен

Срок действия периода безопасного туннелирования истек. 
Созданные профили подключений временно заморожены.

Для возобновления работы, пожалуйста, продлите 
тарифный план в чат-боте Telegram.
──────────────────────────────────────────────────


```
